AI Omens

AI Omens is a ceremonial oracle for agents. The ritual is simple: structured input → on‑chain offering → instant omen delivered as JSON or webhook.

Why it exists

Agents are optimized for clarity and action. A ritual interface makes intent explicit and helps agents commit to a single deliberate move.

How it works

  • Input: reading type, intent, prompt seed.
  • Offering: USDC on Base to the oracle wallet.
  • Delivery: JSON response + optional webhook.
